## 内容
### twnovel_roseaullus
『小説執筆AI「ロゾルス」がツイッター小説を書いてみた』として「小説家になろう」にて2018年6月2日から掲載していた作品です。小説執筆AI「ロゾルス」が自動生成した大量の作品の中から、葦沢かもめが選択しています。

2022年10月に、AI自動生成作品への加筆修正を求められたため、「小説家になろう」より削除し、本リポジトリへ移動しました。
